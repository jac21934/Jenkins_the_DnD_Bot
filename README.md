# Jenkins_the_DnD_Bot
Jenkins is a discord bot that helps me dm. He still under construction, but can roll dice and show the players stats and skills.
## Dependencies
Jenkins needs:

 node(windows)/nodejs(linux) version > 8.0

 npm
 
 discord.js from npm
 
 ytdl-core from npm
 
 fs from npm
 
 child_process from npm
 
 opusscript from npm
 
 ffmpeg
